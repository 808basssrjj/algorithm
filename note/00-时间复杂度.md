## 2. 时间复杂度

推导大O阶:

1. 用常数1代替运行时间中的所有加法常数
2. 只保留最高阶

| 执行次数函数      | 阶        | 非正式术语 |
| ----------------- | --------- | ---------- |
| 12                | O(1)      | 常数阶     |

| 3n^2^+2n+1        | O(n^2^)   | 平方阶     |
| 5log~2~n+20       | O(logn)   | 对数阶     |
| 2n+3nlog~2~n+20   | O(n logn) | n logn     |
